# Rust: «адаптер» без отдельного крейта

Для нативного Rust-приложения достаточно **`synx-core`**:

- `Synx::stringify(&value)` — объект → текст SYNX.
- Обёртка в блок с тремя обратными кавычками и `synx` как язык — по тому же шаблону, что `to_prompt_block` в Python-биндинге.

Отдельный крейт не обязателен: это тонкий слой над уже существующим API.
